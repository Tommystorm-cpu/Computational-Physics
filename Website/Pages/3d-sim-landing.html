Launch System Simulator

The System Simulator allows you to explore fully rendered 3D systems, in your browser.

  • A control panel can be accessed in the top left, which contains many options for you to experiment with, as well as a drop down to load different planetary systems.
  • The time slider at the bottom controls the speed of the simulation, with dedicated buttons for running it at 1 day/second or 1 year/second.
  • You can click/tap on planets to lock onto them instead of using the menu.
  • If you wish to return to this page, there is a "back to website button" in the top right.

Details about how this was made can be found in our Technical Report.
